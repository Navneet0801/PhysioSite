


# PhysioHeal

PhysioHeal is a modern web application designed for physiotherapy clinics to enhance patient engagement, streamline appointment booking, and showcase clinic services and expertise. Built using Next.js and TypeScript, PhysioHeal provides a user-friendly, responsive, and visually appealing platform for both patients and clinic staff.

## Features

- 🏥 **Clinic Information:** Showcase your clinic’s mission, vision, and team.
- 🗓️ **Appointment Booking:** Easy-to-use interface for patients to book appointments online.
- 👨‍⚕️ **Doctors Directory:** List of clinic doctors with detailed profiles.
- 💼 **Service Listings:** Detailed descriptions of all physiotherapy services provided.
- 📞 **Contact and Location Info:** Includes phone, email, and an interactive map.
- 🔗 **Quick Links & Social Media:** Fast navigation to key pages and social media integration.
- 📱 **Responsive Design:** Optimized for all devices.

## Technologies Used

- [Next.js](https://nextjs.org/) (React framework)
- [TypeScript](https://www.typescriptlang.org/)
- [Tailwind CSS](https://tailwindcss.com/) (or your preferred CSS framework)
- [Lucide Icons](https://lucide.dev/) for social and contact icons

## Getting Started

### Prerequisites

- Node.js (v18+ recommended)
- npm or yarn

### Installation

```bash
git clone https://github.com/PrasanBora/PhysioHeal.git
cd PhysioHeal
npm install
# or
yarn install
```

### Running the Development Server

```bash
npm run dev
# or
yarn dev
```

Visit [http://localhost:3000](http://localhost:3000) to view the app.

### Building for Production

```bash
npm run build
npm start
# or
yarn build
yarn start
```

## Folder Structure

```
components/       # React components (e.g., Footer, Header, DoctorCard)
pages/            # Next.js pages (routes)
public/           # Static files and images
styles/           # CSS or Tailwind config
```
